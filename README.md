# Student Persistence Prediction

Machine learning and deep learning framework for predicting student persistence and academic success using the Open University Learning Analytics Dataset (OULAD).

## Dataset
Open University Learning Analytics Dataset (OULAD)

## Models
- Logistic Regression
- Random Forest
- XGBoost
- Neural Network

## Research Questions
1. Can machine learning and deep learning models accurately predict student persistence and academic success?
2. Which demographic, academic, and engagement variables contribute most strongly to student outcomes?
3. Do deep learning models outperform traditional machine learning methods?
4. How might predictive analytics support future intervention strategies in STEM education programs?
